Memory Game
===========

Septembers's entry into the 1 Game Per Month Challenge.

<img src="screenshot.png?raw=true" alt="Memory Game Screenshot" width="512" height="448">

A simple card matching memory game.


Controls
========

 * **D-Pad:** move cursor
 * **B:** Select card

Build Requirements
===================
 * snesdev-common
 * ca65
 * pcx2snes
 * gnu Make
 * python3
 * ucon64
